# Лабораторная работа 1, вариант 7

Flask-приложение принимает изображение, меняет порядок цветовых каналов RGB по выбору пользователя и строит графики для исходного изображения:

- распределение значений каналов R, G, B;
- среднее значение цвета по вертикальной оси;
- среднее значение цвета по горизонтальной оси.

Дополнительно реализована простая CAPTCHA в виде арифметического примера.

## Запуск через Docker

```bash
docker compose up --build lab1-web
```

Открыть приложение: http://localhost:8000

## Тесты

```bash
docker compose run --rm lab1-web pytest tests
```

## Структура

- `app/web.py` - Flask routes и CAPTCHA;
- `app/image_processing.py` - обработка изображений и построение графиков;
- `app/templates/index.html` - HTML-интерфейс;
- `app/static/css/styles.css` - стили;
- `tests/test_app.py` - тесты ключевой логики.
